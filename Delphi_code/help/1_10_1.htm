Меню «Окна» >> Ба чжай (Восемь жилищ)

(кнопка на главном окне программы)

Команда вызывает окно «Ба чжай».


Вид окна «Ба чжай»


Система Ба чжай, представляет собой учение о восьмими секторах пространства. Четыре из них благоприятные для того или иного человека, другие четыре -- нет. Эта система соотносится как с жильем, домом, земельным участком, так и с отдельным человеком. Хорошо, когда благоприятные сектора места обитания человека совпадают с его собственными благоприятными секторами и плохо, когда наоборот.

При нажатии на кнопке «Описание для Дворца», становится доступным подробное описание размещения для дома или человека тех или иных значимых элементов по данному раскладу Ба чжай. См. пример такого описания для Кань-гуа. При нажатии на самом изображении гуа в центре окна, становится доступным описание для гуа.
